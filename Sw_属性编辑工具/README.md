## 选择文件：

## 获得属性：
获得属性可以通过左侧的复选框，选择需要处理的属性名。如果默认不做选择的话，则是输出全部的属性。

## 修改属性：
根据表格所显示的内容，对处理文件的属性名和属性值进行修改。批量修改小技巧：你可以全选表格粘贴到Excel进行刷格式的修改。

## 删除属性：
同获得属性操作类似，你可以通过左侧的复选框，选择需要删除的属性名。如果默认不做选择的话，则是删除全部的属性。
