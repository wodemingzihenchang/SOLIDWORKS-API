##介绍
工程文件主要是以C#的winform模板制作，那你就需要先下载C#相关的编程环境。具体步骤这里不展开。

.sln的项目文件内，主要内容文件包括：
1、solidwork文件类型的cs文件：
    SwModelDoc.cs：文件类
    SwPart.cs零件类
    SwAssembly.cs装配体类
    SwDrawing.cs工程图类
2、File.cs是Windows的文件常规操作类，有文件的打开、txt的读写
3、EditExcel是Excel的读写类
